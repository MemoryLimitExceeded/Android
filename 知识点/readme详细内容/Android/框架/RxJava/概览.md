### 概览

​		RxJava 是 ReactiveX 的 Java VM 实现，而 ReactiveX 是一个可以利用可观察序列和 LINQ 风格查询操作符来编写异步和基于事件的程序，使用 Rx，开发者可以用 Observables 表示异步数据流，用 LINQ 操作符查询异步数据流， 用 Schedulers 参数化异步数据流的并发处理。
