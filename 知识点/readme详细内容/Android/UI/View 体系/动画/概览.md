### 概览

​	动画作为 View 体系中重要的一部分，通过动画来使得 View 能在用户的面前动起来，从而加强用户体验和界面炫酷效果。

#### 目录

- View 动画

  - 补间动画
  - 属性动画

- 逐帧动画

- 布局过渡动画

  ***

- ##### View 动画

  View 动画一般是针对 View 自身显示的动画，分为补间动画和属性动画。

- 逐帧动画

  最简单粗暴的动画，将多组图片按照类似幻灯片的形式快速播放以达到动画的效果。

- 布局过渡动画

  一般用于 Activity、Fragment 等跳转的动画。
